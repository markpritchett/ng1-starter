#Getting Started

## Installing
```
npm install
bower install
```

## Running the app

```
gulp serve
```

## Unit Testing

```
gulp tdd
```

## Deployment Builds

```
gulp build:dev
```
Builds the app and places all files in **./dist** folder

```
gulp build:release
```
Builds the app in release mode, combining and minifying JS and CSS assets and places output in **./dist** folder
